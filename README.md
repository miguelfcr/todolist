# todolist
Lista ToDo Simples

- Baixar os fontes
- Instalar os requirements do frontend e do backend
- Executar "npm run dev" para o frontend (http://localhost:8080)
- Executar o arquivo apptodo.py para o backend

Para facilitar o desenvolvimento e os testes, foi escolhido utilizar o sqlite. 
Assim ao executar o backend, será criado o arquivo chamado "todo.db" onde irá servir como o repositório dos dados.
